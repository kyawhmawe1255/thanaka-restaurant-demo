# Atiri Cafe & Eatery Website Demo

Temporary static website demo for Atiri Cafe & Eatery in Koh Phangan.

## Preview Locally

```bash
python3 -m http.server 8080
```

Then open:

```text
http://127.0.0.1:8080/
```

## Deploy

This site is ready for GitHub Pages. Publish from the repository root and set Pages to deploy from the `main` branch.
